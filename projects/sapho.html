SAPHO

Scalable Architecture for Hardware Optimization

SAPHO is a complete hardware architecture and toolchain designed for processor creation, experimentation, and hardware–software co-design. It bundles the YANC compiler toolchain, the POLARIS IDE, the YAWT Wave Tracer, and the PRISM RTL Viewer.

What SAPHO Includes

SAPHO is a complete hardware architecture and toolchain designed for processor creation, experimentation, and hardware–software co-design.

YANC compiler toolchain (CMM + assembly)
POLARIS cross-platform IDE
YAWT Wave Tracer for signal analysis
PRISM RTL Viewer for processor visualisation
Windows and Linux support

The SAPHO Ecosystem

YANC
Compiler toolchain
POLARIS
Cross-platform IDE
YAWT
Wave Tracer
PRISM
RTL Viewer

SAPHO is designed to lower the barrier to processor design experimentation for undergraduate and graduate students, and is actively used in Computer Architecture courses at UFJF.
