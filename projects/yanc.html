Compiler Toolchain

YANC

Yet Another Compiler

YANC is the compilation backbone of SAPHO, bridging high-level processor description languages and low-level execution. It comprises cmmcomp and asmcomp along with auxiliary tools for the full compilation pipeline.

View on GitHub Part of SAPHO All Projects
C Language Flex Bison GCC Open Source

Toolchain Components

cmmcomp: CMM high-level language compiler
asmcomp: Assembly-level compiler
Written in C with Flex, Bison, and GCC
Fully integrated into the SAPHO ecosystem
