AURORA

Windows IDE for the SAPHO Ecosystem

AURORA was a powerful, Windows-native IDE built with Electron for the SAPHO ecosystem. It supported CMM programming, Icarus Verilog simulation, GTKWave waveform visualisation, and the PRISM RTL Viewer. With 25 releases (latest v4.27.10), AURORA played a foundational role before being succeeded by POLARIS.

Legacy Project — succeeded by POLARIS
AURORA is no longer actively developed. All new development happens in POLARIS, which is cross-platform, lighter, and feature-compatible. AURORA's GitHub archive remains available for reference.

AURORA Features

CMM programming environment
Icarus Verilog simulation integration
GTKWave waveform visualisation
PRISM RTL Viewer
25 releases — latest v4.27.10

If you are starting a new SAPHO-based project, we recommend using POLARIS instead. It runs on both Linux and Windows, starts in under one second, and uses significantly less memory than AURORA's Electron runtime.
